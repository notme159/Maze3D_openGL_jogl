open gl (jogl), java se, small 3d graphics maze
__caution__ lots of blinking colors, watch out if you have epilepsy!


import libraries to project depending on your operating system http://www3.ntu.edu.sg/home/ehchua/programming/opengl/JOGL2.0.html specifically section 2. Setting Up JOGL 2.0_rc8 - Step 1 and Step 2: Setup JOGL
e.g. win64:
download zip or clone maze3d, make new java project with maze3d source, add 4 libraries from this http://jogamp.org/deployment/jogamp-current/archive/jogamp-all-platforms.7z
specifically gluegen-rt.jar gluegen-rt-natives-windows-amd64.jar jogl-all.jar jogl-all-natives-windows-amd64.jar
run in eclipse AppMaze3D.java

controls: mouse, WSAD. Experimental (probably buggy): E, Q, hold left mouse button and look around
